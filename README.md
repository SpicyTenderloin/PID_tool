## PID Simulation Tool
This Simulink and MATLAB Live Script models a PID controller governing the altitude of a quadcopter. The tool is designed to provide students with an interactive environment to develop an understanding of control systems and PID control.

# Features
The tool currently includes:
A Simulink model representing the system as a block diagram
A MATLAB Live Script for interactive parameter tuning
Real-time visualisation of step response and system behaviour
Purpose

# This tool aims to:
Demonstrate the effect of proportional, integral, and derivative control
Provide hands-on experience with controller tuning
Bridge the gap between theoretical control concepts and practical UAV systems
